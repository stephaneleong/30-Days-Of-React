# What is React?
React is a JavaScript library for building a reusable user interface(UI).

# What is a library?
A programming library is a collection of prewritten code that programmers can use to optimize tasks.

# What is a single page application?
It is application with only one web page and instead changes by dynamically rewriting the current web page.

# What is a component ?
In a React app, every piece of UI is a component.

# What is the latest version of React?
18.2.0

# What is DOM?
it stands for Document Object Model, it is an interface that presents each HTML element as an object.

# What is React Virtual DOM?
VirtualDOM in React is a lightweight copy of the actual DOM(a virtual representation of the DOM) used to optimize the updates on web page. So for every object that exists in the original DOM, there is an object for that in React Virtual DOM

#  What does a web application or a website(composed of) have?
A web application is composed of a frontend, backend, networking and security.

Exercises: JSX
What is an HTML element?
An HTML element consists of a start tag, content, and an end tag. For example, <p>Hello World</p> is a paragraph element.

How to write a self closing HTML element?
Self-closing tags, also known as void elements, do not have content or an end tag. Examples include <br>, <img>, and <input />.

What is an HTML attribute? Write some of them
HTML attributes provide additional information about HTML elements.
The entire list of HTML attributes can found here --> https://www.w3schools.com/tags/ref_attributes.asp
some of these attributes are such as width, height, onclick, id, etc.

What is JSX?
JSX stands for Javascript XML, this allows us to write HTML element with Javascript Code.
What is babel?
Babel is a toolchain that is mainly used to convert ECMAScript 2015+ code into a backwards compatible version of JavaScript in current and older browsers or environments.
Babel can convert JSX syntax! Check out our React preset to get started. Use it together with the babel-sublime package to bring syntax highlighting to a whole new level.

What is a transpiler?
transpiler also convert an input source code written in some language into an output code in a different language, that has a similar level of abstraction as compared to the source code.
